- Keep the architecture understandable before adding abstractions.
- Prefer deterministic automation, typed contracts, and testable integration boundaries.
- Treat UX, compliance, localization, and operations as part of the product, not afterthoughts.
- Ship in small, verifiable increments with clear rollback paths.
